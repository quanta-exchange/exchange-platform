- Healthchecks pass within reasonable time
- Services are reachable via localhost ports documented

- Integration: produce/consume a test topic

## Observability
- Dashboard: local OTel collector receives traces
- Metrics: broker up, db up

## Rollback / Recovery
- If compose breaks, revert changes and restore last known-good compose
